# Portfolio+ ◈

A sleek, professional portfolio builder for creators. Build, manage, and showcase your work with a modern glassmorphic interface and secure cloud storage.

## 🚀 Features
- **Visual Editor**: Drag-and-drop styled project management.
- **Secure Auth**: JWT-based login/signup system.
- **Cloud Database**: Powered by MongoDB Atlas.
- **Responsive Design**: Flawless experience across all devices.
- **Vercel Ready**: Optimized for serverless deployment.

## 🛠 Tech Stack
- **Backend**: Node.js & Express
- **Database**: MongoDB (Mongoose)
- **Frontend**: Vanilla JS, HTML5, CSS3
- **Security**: JWT, Bcrypt, Helmet, Rate Limiting

## ⚙️ Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file with:
   - `MONGO_URI`
   - `JWT_SECRET`
   - `CLIENT_URL`
4. Run locally: `npm run dev`
